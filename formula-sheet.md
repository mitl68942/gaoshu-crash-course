## 1. 多元函数微分学

### 1.1 偏导数
| 考点 | 公式 | 说明 |
|------|------|------|
| 🔥 一阶偏导 | ∂z/∂x：对 x 求导，y 视为常数 | 把不是该变量的全当常数 |
| ⚠️ 二阶混合偏导 | ∂²z/∂x∂y = ∂²z/∂y∂x | 连续条件下相等，先求哪个都行 |
| 🔥 链式法则 | dz/dt = (∂z/∂x)·(dx/dt) + (∂z/∂y)·(dy/dt) | 全导数，画树形图 |

### 1.2 全微分
| 考点 | 公式 |
|------|------|
| 🔥 全微分 | dz = (∂z/∂x)dx + (∂z/∂y)dy |
| 近似计算 | Δz ≈ dz = f_x·Δx + f_y·Δy |

### 1.3 方向导数与梯度
| 考点 | 公式 |
|------|------|
| 方向导数 | ∂f/∂l = ∇f · l₀ = f_x·cosα + f_y·cosβ |
| 🔥 梯度 | ∇f = (f_x, f_y, f_z) → 方向导数最大的方向 |
| 梯度模 | \|∇f\| = √(f_x² + f_y²) |

### 1.4 隐函数求导
| 类型 | 公式 |
|------|------|
| F(x,y)=0 求 dy/dx | dy/dx = -F_x/F_y |
| F(x,y,z)=0 求偏导 | ∂z/∂x = -F_x/F_z，∂z/∂y = -F_y/F_z |

### 1.5 极值与最值
| 考点 | 公式/步骤 |
|------|-----------|
| 驻点 | f_x=0 且 f_y=0 |
| 🔥 判别式 | A=f_xx，B=f_xy，C=f_yy，Δ=AC-B² |
| Δ>0 且 A>0 | 极小值 |
| Δ>0 且 A<0 | 极大值 |
| Δ<0 | 不是极值（鞍点） |
| Δ=0 | 无法判定，另寻他法 |
| ⚠️ 条件极值 | 拉格朗日乘数法：L(x,y,λ)=f(x,y)+λ·φ(x,y)，解 L_x=0，L_y=0，L_λ=0 |

## 2. 重积分

### 2.1 二重积分计算
| 考点 | 步骤 |
|------|------|
| 🔥 画区域 + 定限 | 先看清楚积分区域形状，决定 dxdy 还是 dydx |
| X型区域 | ∬_D f(x,y)dxdy = ∫[a,b]dx∫[y₁(x),y₂(x)]f(x,y)dy |
| Y型区域 | ∬_D f(x,y)dxdy = ∫[c,d]dy∫[x₁(y),x₂(y)]f(x,y)dx |

### 2.2 坐标变换
| 变换 | 公式 | 雅可比 \|J\| |
|------|------|-------------|
| 🔥 极坐标 | x = r·cosθ，y = r·sinθ | r |
| 广义极坐标 | x = a·r·cosθ，y = b·r·sinθ | ab·r |
| 一般变换 | x = x(u,v)，y = y(u,v) | \|∂(x,y)/∂(u,v)\| |

**⚠️ 极坐标定限口诀**：先画区域，θ 从哪转到哪，r 从什么函数到什么函数

### 2.3 三重积分
| 方法 | 适用场景 |
|------|----------|
| 🔥 投影法（先一后二） | 先 z 再 xy：∫dxdy∫[z₁,z₂]f dz |
| 截面法（先二后一） | 先 xy 再 z：∫dz∬[D(z)]f dxdy |
| 🔥 柱坐标 | x=rcosθ，y=rsinθ，z=z，\|J\|=r |
| 🔥 球坐标 | x=r·sinφ·cosθ，y=r·sinφ·sinθ，z=r·cosφ，\|J\|=r²·sinφ |

### 2.4 应用
| 应用 | 公式 |
|------|------|
| 面积 | A = ∬_D 1 dxdy |
| 体积 | V = ∬_D [z上(x,y) - z下(x,y)] dxdy 或 ∭_Ω 1 dxdydz |
| 曲面积分 | 高斯公式（见下文） |

## 3. 无穷级数

### 3.1 常数项级数审敛法
| 方法 | 适用条件 | 判别规则 |
|------|----------|----------|
| 必要条件 | 所有级数 | 若 lim u_n ≠ 0 → 发散 |
| 🔥 比值审敛法 | u_n > 0 | ρ = lim u_{n+1}/u_n；ρ<1 收敛，ρ>1 发散，ρ=1 换方法 |
| 根值审敛法 | u_n > 0 | ρ = lim ⁿ√u_n；同上 |
| 🔥 p-级数 | Σ 1/n^p | p>1 收敛，p≤1 发散 |
| 比较审敛法 | 找一个已知级数 | u_n ≤ v_n 且 Σv_n 收敛 → Σu_n 收敛 |
| ⚠️ 交错级数 | (-1)^n u_n | u_n 单调递减趋于 0 → 条件收敛 |
| 绝对收敛 | Σ\|u_n\| 收敛 | 绝对收敛 ⇒ 原级数收敛 |

### 3.2 幂级数
| 考点 | 公式/步骤 |
|------|-----------|
| 🔥 收敛半径 R | ρ = lim \|a_{n+1}/a_n\|，R = 1/ρ（ρ=0 则 R=∞；ρ=∞ 则 R=0）|
| 🔥 收敛域 | [-R, R] 或 (-R, R) 或 [x₀-R, x₀+R]（平移后），端点单独判断 |
| 和函数 S(x) | 逐项求导/逐项积分 |
| 🔥 展开为幂级数 | 记牢 5 个基本展开式！ |

**5 个必背展开式（考场默写）：**
```
1. e^x     = Σ[x^n/n!]            (-∞,+∞)
2. sin x   = Σ[(-1)^n·x^{2n+1}/(2n+1)!]  (-∞,+∞)
3. cos x   = Σ[(-1)^n·x^{2n}/(2n)!]      (-∞,+∞)
4. 1/(1-x) = Σ[x^n]              (-1,1)
5. ln(1+x) = Σ[(-1)^{n-1}·x^n/n] (-1,1]
```

### 3.3 傅里叶级数
| 考点 | 公式 |
|------|------|
| 🔥 周期 2π | f(x) = a₀/2 + Σ[a_n·cos(nx) + b_n·sin(nx)] |
| a₀ | a₀ = (1/π)∫[-π,π]f(x)dx |
| a_n | a_n = (1/π)∫[-π,π]f(x)cos(nx)dx |
| b_n | b_n = (1/π)∫[-π,π]f(x)sin(nx)dx |
| 周期 2l | 用 T=2l 代换，cos(nπx/l)，sin(nπx/l) |
| ⚠️ 间断点 | 收敛到 [f(x⁺)+f(x⁻)]/2（左右极限平均值） |

## 4. 常微分方程

### 4.1 一阶微分方程
| 类型 | 识别特征 | 解法 |
|------|----------|------|
| 🔥 可分离变量 | dy/dx = g(x)·h(y) | 分离：dy/h(y) = g(x)dx，两边积分 |
| 🔥 一阶线性 | y' + P(x)y = Q(x) | 通解：y = e^{-∫Pdx}[∫Q·e^{∫Pdx}dx + C] |
| 齐次方程 | dy/dx = φ(y/x) | 令 u=y/x，化成可分离变量 |
| 伯努利方程 | y' + P(x)y = Q(x)·y^n | 令 z = y^{1-n}，化为一阶线性 |

### 4.2 二阶常系数线性微分方程
| 类型 | 形式 | 解法 |
|------|------|------|
| 🔥 齐次 | y'' + py' + qy = 0 | 特征方程 r²+pr+q=0 |
|  | Δ>0：r₁≠r₂（实） | y = C₁e^{r₁x} + C₂e^{r₂x} |
|  | Δ=0：r₁=r₂=r | y = (C₁+C₂x)e^{rx} |
|  | Δ<0：r=α±βi | y = e^{αx}(C₁cosβx+C₂sinβx) |

### 4.3 二阶非齐次（🔥 必考）
| f(x) 形式 | 特解 y* 设的形式 |
|-----------|-----------------|
| P_m(x)（m 次多项式） | ⚠️ 看 0 是不是特征根：不是→Q_m(x)；是单根→x·Q_m；是重根→x²·Q_m |
| 🔥 e^{λx}·P_m(x) | ⚠️ 看 λ 是不是特征根：不是→Q_m·e^{λx}；是单根→x·Q_m·e^{λx}；是重根→x²·Q_m·e^{λx} |
| 🔥 e^{αx}[P_lcosβx+P_m sinβx] | ⚠️ 看 α±βi 是不是特征根：不是→e^{αx}[R_n cosβx+S_n sinβx]；是→x 乘同上 |

**⚠️ 非齐次通解 = 齐次通解 + 非齐次特解**（结构不能忘！）

### 4.4 其他
| 类型 | 技巧 |
|------|------|
| 可降阶 y''=f(x,y') | 令 p=y'，化为一阶 |
| 可降阶 y''=f(y,y') | 令 p=y'，则 y''=p·dp/dy |
