## 模块 1：多元函数微分学

### 例题 1.1：隐函数求偏导（🔥 必考）

**【考法】** 给一个 F(x,y,z)=0 形式的方程，求 ∂z/∂x 或 ∂z/∂y

**【题干】** 设方程 x² + y² + z² - 4z = 0 确定隐函数 z = z(x,y)，求 ∂z/∂x 和 ∂²z/∂x∂y。

**【🚫 常见踩坑】**
- 忘了 F_z 在分母，符号容易搞错
- 二阶偏导要对一阶结果再次求导，注意 z 本身也是 x 的函数

**【✅ 考场上这么写：**

```
令 F(x,y,z) = x² + y² + z² - 4z

Step 1：求偏导数
  F_x = 2x
  F_y = 2y
  F_z = 2z - 4

Step 2：一阶偏导
  ∂z/∂x = -F_x/F_z = -2x/(2z-4) = -x/(z-2)  ← 化简到最简
  ∂z/∂y = -F_y/F_z = -2y/(2z-4) = -y/(z-2)

Step 3：二阶混合偏导（对 ∂z/∂x 再对 y 求导）
  ∂²z/∂x∂y = ∂/∂y [-x/(z-2)]
           = -x · [-1/(z-2)²] · ∂z/∂y     ← 复合函数求导
           = x/(z-2)² · [-y/(z-2)]        ← 代入 ∂z/∂y
           = -xy/(z-2)³
```
】

### 例题 1.2：条件极值（拉格朗日乘数法）（🔥 必考）

**【考法】** 给一个约束条件，求多元函数的最值（长方体最大体积/最短距离等应用题）

**【题干】** 求函数 f(x,y) = xy 在条件 x + y = 1 下的极值。

**【🚫 常见踩坑】**
- 忘了约束条件是 φ(x,y)=0 的形式（要移项）
- 解方程组漏解

**【✅ 考场上这么写：**

```
Step 1：构造拉格朗日函数
  L(x,y,λ) = xy + λ(x + y - 1)    ← 约束条件移项为 x+y-1=0

Step 2：求偏导并令其为零
  L_x = y + λ = 0      ①
  L_y = x + λ = 0      ②
  L_λ = x + y - 1 = 0  ③

Step 3：解方程组
  由①：y = -λ
  由②：x = -λ
  由③：(-λ) + (-λ) = 1 → λ = -1/2
  所以 x = y = 1/2

Step 4：计算极值
  f(1/2, 1/2) = 1/4

答：在 (1/2, 1/2) 处取得极值 1/4。
（至于是极大还是极小，结合几何意义判断）
```

## 模块 2：重积分

### 例题 2.1：二重积分 + 极坐标变换（🔥 必考）

**【考法】** 积分区域是圆或圆的一部分，换极坐标后计算

**【题干】** 计算 ∬_D e^{-x²-y²} dxdy，其中 D 为 x² + y² ≤ a²。

**【🚫 常见踩坑】**
- 换极坐标忘了乘 r（雅可比行列式）
- θ 和 r 的上下限定错
- 积分时忘了 r 的存在导致算不出

**【✅ 考场上这么写：**

```
Step 1：换极坐标
  x = r·cosθ，y = r·sinθ
  x² + y² = r²
  dxdy = r dr dθ              ← ⚠️ 千万别漏这个 r！

Step 2：确定积分限（先画图）
  D：x²+y² ≤ a² → 0 ≤ r ≤ a，0 ≤ θ ≤ 2π

Step 3：代入计算
  ∬_D e^{-x²-y²} dxdy
  = ∫[0,2π] dθ ∫[0,a] e^{-r²} · r dr    ← r 乘进来了

  内层积分：∫[0,a] r·e^{-r²} dr
  令 t = r²，dt = 2r dr，r dr = dt/2
  = ∫[0,a²] e^{-t} · dt/2 = (1/2)[-e^{-t}][0,a²] = (1/2)(1 - e^{-a²})

  外层积分：∫[0,2π] (1/2)(1 - e^{-a²}) dθ
  = (1/2)(1 - e^{-a²}) · 2π = π(1 - e^{-a²})

答：∬_D e^{-x²-y²} dxdy = π(1 - e^{-a²})
```
】

### 例题 2.2：三重积分（柱坐标）（🔥 常考）

**【考法】** 积分区域是柱体、圆锥或旋转体，换柱坐标

**【题干】** 计算 ∭_Ω z dxdydz，Ω 由 z = √(x²+y²) 和 z = h（h>0）围成。

**【🚫 常见踩坑】**
- 柱坐标的 |J| = r 别忘了
- z 的上下限：下界是锥面 √(x²+y²) = r，上界是平面 z=h

**【✅ 考场上这么写：**

```
Step 1：换柱坐标
  x = r·cosθ，y = r·sinθ，z = z
  dxdydz = r dr dθ dz

Step 2：确定积分限
  Ω：锥面 z = r 到 平面 z = h
  r 的范围：从 0 到 h（在 z=h 处 r=h）
  θ：0 到 2π

Step 3：计算
  ∭_Ω z dxdydz
  = ∫[0,2π] dθ ∫[0,h] r dr ∫[r,h] z dz

  最内层：∫[r,h] z dz = (1/2)(h² - r²)

  中层：∫[0,h] r · (1/2)(h² - r²) dr
  = (1/2)[h²·r²/2 - r⁴/4][0,h]
  = (1/2)(h⁴/2 - h⁴/4) = (1/2)(h⁴/4) = h⁴/8

  外层：∫[0,2π] h⁴/8 dθ = (h⁴/8)·2π = πh⁴/4

答：∭_Ω z dxdydz = πh⁴/4
```
】

## 模块 3：无穷级数

### 例题 3.1：幂级数收敛域（🔥 必考）

**【考法】** 给一个幂级数，求收敛半径和收敛域

**【题干】** 求幂级数 Σ[n=1,∞] (x-2)^n / (n·3^n) 的收敛域。

**【🚫 常见踩坑】**
- 忘了它是"关于 (x-2) 的幂级数"，收敛中心在 x=2
- 端点忘记单独判断（这是最常见扣分点！）

**【✅ 考场上这么写：**

```
Step 1：求收敛半径
  令 t = x-2，级数变为 Σ t^n/(n·3^n)

  用比值法：a_n = 1/(n·3^n)
  ρ = lim[n→∞] |a_{n+1}/a_n|
    = lim[n→∞] [1/((n+1)·3^{n+1})] / [1/(n·3^n)]
    = lim[n→∞] n·3^n / [(n+1)·3^{n+1}]
    = lim[n→∞] n / [3(n+1)] = 1/3

  收敛半径 R = 1/ρ = 3

Step 2：确定收敛区间（暂不考虑端点）
  |t| < 3 → |x-2| < 3 → -3 < x-2 < 3 → -1 < x < 5

Step 3：判断端点 ⚠️
  ① x = -1（t = -3）：
     Σ(-1)^n·3^n/(n·3^n) = Σ(-1)^n/n → 交错调和级数，条件收敛 ✓

  ② x = 5（t = 3）：
     Σ 3^n/(n·3^n) = Σ 1/n → 调和级数，发散 ✗

答：收敛域为 [-1, 5)
```
】

### 例题 3.2：函数展开成幂级数（🔥 常考）

**【考法】** 给一个函数，展开成 (x-a) 的幂级数

**【题干】** 将 f(x) = 1/(1+x²) 展开成 x 的幂级数，并求收敛区间。

**【🚫 常见踩坑】**
- 不会变形到基本展开式的形式
- 忘了交代收敛区间

**【✅ 考场上这么写：**

```
Step 1：利用已知展开式
  已知：1/(1-u) = Σ[n=0,∞] u^n，|u| < 1

Step 2：变量代换
  f(x) = 1/(1+x²) = 1/(1-(-x²))
  令 u = -x²，则：
  f(x) = Σ[n=0,∞] (-x²)^n = Σ[n=0,∞] (-1)^n · x^{2n}

Step 3：收敛条件
  |u| < 1 → |x²| < 1 → |x| < 1

答：1/(1+x²) = Σ[n=0,∞] (-1)^n·x^{2n}，收敛区间 (-1, 1)
```
】

## 模块 4：常微分方程

### 例题 4.1：一阶线性微分方程（🔥 必考）

**【考法】** 给一个 y' + P(x)y = Q(x) 形式，求通解

**【题干】** 解微分方程 y' + 2xy = x。

**【🚫 常见踩坑】**
- ∫Pdx 算错导致后面的 e^∫Pdx 全错
- 通解公式中的积分常数 C 别忘记

**【✅ 考场上这么写：**

```
Step 1：识别类型
  这是 y' + P(x)y = Q(x) 形式，其中 P(x)=2x，Q(x)=x

Step 2：套通解公式
  y = e^{-∫Pdx} · [∫ Q·e^{∫Pdx} dx + C]

  先算 ∫Pdx = ∫2x dx = x²

Step 3：代入
  y = e^{-x²} · [∫ x·e^{x²} dx + C]

  注意到 ∫ x·e^{x²} dx = (1/2)∫ e^{x²} d(x²) = (1/2)e^{x²}

  y = e^{-x²} · [(1/2)e^{x²} + C]
    = 1/2 + C·e^{-x²}

答：通解为 y = 1/2 + C·e^{-x²}
```
】

### 例题 4.2：二阶常系数非齐次（🔥 必考）

**【考法】** 给 y'' + py' + qy = f(x)，求通解（考场上 10 分大题）

**【题干】** 求 y'' - 3y' + 2y = e^{3x} 的通解。

**【🚫 常见踩坑】**
- 只求了特解忘了加齐次通解
- 设特解 y* 时，没检查 λ 是不是特征根
- 代入求待定系数时代错

**【✅ 考场上这么写：**

```
Step 1：先求齐次通解
  特征方程：r² - 3r + 2 = 0
  (r-1)(r-2) = 0 → r₁=1，r₂=2

  齐次通解：y_c = C₁e^x + C₂e^{2x}

Step 2：设特解 ⚠️
  右端 f(x) = e^{3x}，λ=3
  检查：λ=3 不是特征根（特征根是 1 和 2）
  → 设 y* = A·e^{3x}

Step 3：代入求 A
  y*' = 3A·e^{3x}
  y*'' = 9A·e^{3x}

  代入原方程：
  9A·e^{3x} - 3·3A·e^{3x} + 2·A·e^{3x} = e^{3x}
  (9A - 9A + 2A)·e^{3x} = e^{3x}
  2A = 1 → A = 1/2

  y* = (1/2)·e^{3x}

Step 4：通解 = 齐次 + 特解
  y = C₁e^x + C₂e^{2x} + (1/2)e^{3x}

答：通解为 y = C₁e^x + C₂e^{2x} + (1/2)e^{3x}
```
】

## 考前综合自测题

**【题干】** 求曲面 z = x² + y² 在点 (1,1,2) 处的切平面方程和法线方程。

**【✅ 考场写法：**

```
令 F(x,y,z) = x² + y² - z = 0

偏导：F_x=2x，F_y=2y，F_z=-1
在 (1,1,2)：F_x=2，F_y=2，F_z=-1

切平面：2(x-1) + 2(y-1) - (z-2) = 0
       → 2x + 2y - z - 2 = 0

法线：(x-1)/2 = (y-1)/2 = (z-2)/(-1)
```
】
